# TEMPLATE REACT

Template con l'utilizzo di React (Vite.js) con gia' configurato ZOHO (libreria CRMPartnersLib), Tailwind CSS. Buon widget!

## Comandi

- installa le dipendenze `npm i`
- locale: `npm run dev`

N.B.: avviare dentro Zoho CRM
